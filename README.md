# Laskin

*Laskin* is a reverse polish notation calculator / programming language
inspired by [Forth], [RPL] and [Plorth].

## Requirements

* [GNU Multiple Precision Arithmetic Library]
* [GNU Readline]
* [CMake]
* C++17 compatible C++ compiler

## How to compile

```bash
mkdir build && cd build && cmake .. && make
```
